# 📊 Analisador de Sentimentos Profissional (NLP)

Este projeto é uma ferramenta de IA de ponta a ponta que realiza a análise de sentimentos em textos e arquivos CSV (como exportações de comentários do YouTube). Desenvolvido com uma arquitetura limpa e escalável.

## 🚀 Funcionalidades
- **Análise em Tempo Real:** Digite uma frase e receba a classificação instantânea.
- **Processamento em Lote:** Faça upload de arquivos CSV e analise milhares de linhas de uma vez.
- **Limpeza de Dados (Preprocessing):** Remoção automática de links, menções (@) e ruídos de redes sociais.
- **Dashboard Visual:** Gráficos interativos gerados com Streamlit e Plotly.

## 🛠️ Tecnologias
- **Linguagem:** Python 3.12
- **IA/ML:** Hugging Face Transformers (Modelo: BERTweet)
- **Interface:** Streamlit
- **Gestão de Dados:** Pandas
- **Ambiente:** UV (Modern Python Package Manager)

## 📦 Como Instalar
1. Certifique-se de ter o **Python 3.10+** instalado.
2. Baixe a pasta do projeto.
3. Clique duas vezes no arquivo `instalar.bat`. Ele criará o ambiente e baixará as dependências automaticamente.

## 📈 Como Usar
1. Clique duas vezes em `iniciar.bat`.
2. O dashboard abrirá no seu navegador padrão.
3. Escolha entre "Análise Única" ou "Upload de CSV".
